Instructions

To use this page properly, you'll need to edit the markup to match your style guidelines.

The ones provided here are based on the article - Modular Layout Systems by Jason Santa Maria.

The goal of this page is to help you style content, such as blog posts, where styling will be done by an author. In this situation, classes for style is the best way to ensure consistency. This page should then be used as a reference for your content authors.
